# PETOPIA - Vet & Pet Appointment System

A comprehensive web application for booking veterinary appointments and managing pet care services.

## Features

### For Pet Owners:
- **Book Appointments**: Easy-to-use booking system with step-by-step process
- **Select Veterinarian**: Choose from available vets
- **Date & Time Selection**: Interactive calendar and time slot picker
- **My Bookings**: View and manage all your appointments
- **Booking Details**: Detailed view of each appointment with status tracking

### For Veterinarians:
- **Vet Dashboard**: Comprehensive dashboard to manage appointments
- **Appointment Management**: View, confirm, or reject appointments
- **Weekly Schedule**: Visual calendar grid to manage availability
- **Time Slot Management**: Add, edit, enable/disable time slots
- **Status Tracking**: Track appointment statuses (Awaiting Confirmation, Confirmed, Cancelled)

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Local Storage (for data persistence)

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/DevAhmad-CS/petopia-vet-booking.git
```

2. Open `index.html` in your web browser

3. No installation or build process required - it's a static website!

## Usage

### For Pet Owners:
1. Navigate to the booking page
2. Select a veterinarian
3. Choose date and time
4. Fill in your details and pet information
5. Confirm your booking

### For Veterinarians:
1. Login through the vet login page
2. Access your dashboard
3. Manage appointments and availability
4. Update appointment statuses
